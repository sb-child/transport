# 神圣跨性别帝国护照生成器

![img](example.png)

## 准备您的头像

请提前将自己的头像剪切为`4:5`格式，如需上方留有空白请提前预留。

![image](https://user-images.githubusercontent.com/96931510/191689366-ff74e9d0-d097-4eac-a535-7877eb1f93dd.png)

## 安装依赖
```bash
pip3 install pillow qrcode
```

## 生成属于您的护照
```bash
python3 main.py
```

您的护照将会保存在`out.png`中
